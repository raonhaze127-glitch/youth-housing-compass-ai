# 청년주거나침반 AI

AI가 청년의 조건에 맞는 공공주택과 주거지원 정책을 추천하는 대화형 주거 길잡이 서비스입니다.

## 문제

청년들은 일자리와 교육 기회를 찾아 수도권으로 이동하지만, 높은 전월세 부담과 복잡한 공공주거 정보로 인해 자신에게 맞는 정책을 찾기 어렵습니다. LH, SH, 마이홈, 지자체 등에 정보가 흩어져 있고 공고문은 자격요건과 신청 절차가 복잡해 정보 비대칭이 발생합니다.

## 해결

청년주거나침반 AI는 사용자의 자연어 입력에서 지역, 나이, 무주택 여부, 소득 수준, 관심 유형을 추출하고 구조화된 주거지원 데이터와 비교해 신청 가능성이 높은 사업을 추천합니다.

## MVP 기능

- 자연어 기반 사용자 조건 입력
- 규칙 기반 조건 추출: 나이, 지역, 무주택 여부, 소득 수준, 관심 유형
- 공공주택 및 주거지원 샘플 데이터 매칭
- 추천 결과 3개 카드 출력
- 추천 이유, 신청기간, 지원내용, 원문 링크 제공
- 추후 OpenAI 또는 Gemini API 연결을 고려한 로직 분리

## 기술 스택

- Next.js
- TypeScript
- JSON 기반 초기 데이터
- Vercel 배포 가능 구조

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 현재 범위

회원가입, 로그인, 알림, 지도 기능은 포함하지 않습니다. 공모전 제출 전까지는 실제 동작하는 추천 MVP 완성에 집중합니다.
