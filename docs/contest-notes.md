# 공모전 제출 메모

## 시연에서 보여줄 범위

- 사용자의 한 문장 입력
- 규칙 기반 조건 추출
- 샘플 공고와의 매칭
- 접수중, 모집예정, 마감 공고의 구분과 정렬
- 추천 이유와 원문 확인 경로

## 과장하지 않을 표현

- 실시간이라고 과장하지 않고, 매일 수집·검증해 저장한 공고라고 설명합니다.
- 실제 자격을 확정한다고 설명하지 않습니다.
- 독립적으로 동작하는 다중 AI 에이전트가 완성됐다고 설명하지 않습니다.
- 지속적으로 스스로 학습한다고 설명하지 않습니다.

## 확장성 표현

“향후 상담 이력, 사용자 피드백, 신규 공고 데이터를 반영해 추천 품질을 고도화할 수 있는 구조”라고 설명합니다.

## 필수 안내

현재 기본 데이터는 LH·SH·GH에서 수집하고 공고문을 구조화한 저장 실공고입니다. 수집 실패에 대비한 출품용 샘플은 최종 폴백으로만 사용합니다. 추천 결과는 참고용이며 실제 신청 전 기관의 최신 공고문과 자격요건을 확인해야 합니다.
