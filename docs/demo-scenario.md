# 시연 시나리오

## 기본 시나리오

입력:

```text
서울 사는 28세 무주택 직장인인데 월세 부담이 커요
```

예상 추출 조건:

- 지역: 서울
- 나이: 만 28세
- 무주택: 예
- 유형: 청년
- 관심: 월세

예상 추천:

1. 서울시 청년월세지원
2. 청년안심주택 입주자 모집
3. LH 청년매입임대주택

## 신혼부부 시나리오

입력:

```text
서울 강서구 33세 신혼부부인데 공공임대주택을 알아보고 싶어요
```

확인 포인트:

- 신혼부부 유형을 추출합니다.
- 신혼부부 매입임대주택이 추천 후보로 올라옵니다.
- 전국 단위 사업은 지역 조건에서 통과됩니다.

## 전세지원 시나리오

입력:

```text
경기 거주 31세 무주택 청년이고 월소득 230만원이라 전세보증금 지원을 찾고 있어요
```

확인 포인트:

- 경기 지역과 전세 관심 유형을 추출합니다.
- 소득 수준을 낮음으로 추출합니다.
- 경기도 청년 전세보증금 지원이 높은 점수를 받습니다.
