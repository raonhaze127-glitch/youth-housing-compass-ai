# 로드맵

## 현재: 출품용 MVP

- 규칙 기반 사용자 조건 추출
- 구조화된 샘플 공고 데이터
- 모집 상태 자동 계산 및 정렬
- 추천 카드와 모바일 화면

## 다음 단계: 데이터 기반 강화

- k-apt-alert 기반 실데이터 수집 구조 검토
- 기관별 원문 저장과 공통 스키마 정규화
- 공고 원문 분석 및 출처 검증
- 샘플 데이터와 실데이터의 명확한 구분

## 이후 단계: 사용자 경험 고도화

- RAG 기반 공고 질의응답
- 상담 이력과 명시적 사용자 피드백을 활용한 추천 평가
- 신규 공고 알림 및 카카오톡 연계
- 모바일 앱 확장

각 단계는 데이터 정확도와 사용자 검증을 선행합니다. 자동 학습을 전제하지 않으며, 수집 데이터와 피드백을 검토해 추천 품질을 개선합니다.
