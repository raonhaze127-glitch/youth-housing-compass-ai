# 서비스 기획서

## 서비스명

청년주거나침반 AI

## 한 줄 소개

AI가 청년의 조건에 맞는 공공주택과 주거지원 정책을 찾아주는 대화형 주거 길잡이 서비스

## 핵심 메시지

길을 찾을 때 지도를 사용하듯, 청년이 자신에게 맞는 주거 정책과 공공주택을 찾을 때 사용하는 AI 나침반입니다.

## 대상 사용자

- 만 19~39세 청년
- 사회초년생
- 신혼부부
- 무주택자
- 공공주택 또는 주거비 지원사업을 찾는 사람

## MVP 사용자 흐름

1. 사용자가 자연어로 현재 상황을 입력합니다.
2. 서비스가 지역, 나이, 무주택 여부, 소득 수준, 관심 유형을 추출합니다.
3. 구조화된 공고 데이터와 조건을 비교합니다.
4. 추천 결과 3개를 카드로 보여줍니다.
5. 사용자는 추천 이유와 원문 링크를 확인합니다.

## 공모전 평가 대응

사회문제는 청년 주거 불안과 공공주거 정보 비대칭입니다. AI 필요성은 복잡한 공고문 분석, 사용자 조건 구조화, 쉬운 설명 제공에 있습니다. MVP는 기존 공고 수집 구조와 JSON 데이터를 활용해 2~3주 안에 구현 가능한 범위로 제한합니다.
