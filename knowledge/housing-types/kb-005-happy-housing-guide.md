# KB-005 행복주택 가이드

- KB ID: `KB-005`
- 주제: 행복주택 상담·자격진단 가이드
- 기준일: 2026-07-03
- 사용 범위: Policy Agent, Eligibility Agent, Recommendation Agent, Verification Agent
- 우선순위: 모집공고 원문 > LH 청약플러스 최신 안내 > 이 문서

## 한 줄 정의

행복주택은 젊은 계층의 주거안정을 위해 대중교통이 편리하거나 직장·학교와 가까운 곳에 시중 시세보다 저렴하게 공급하는 공공임대주택이다.

청주나 상담에서는 “대학생, 청년, 신혼부부, 한부모가족 등 생애 초기 계층과 고령자·주거급여수급자의 주거비 부담을 낮추기 위한 공공임대”로 설명한다.

## 핵심 특징

- 국가 재정과 국민주택기금 지원을 받아 공급하는 공공임대주택이다.
- 공급대상은 대학생, 청년, 신혼부부, 한부모가족, 고령자, 주거급여수급자 등이다.
- 임대조건은 시중 시세의 60~80% 수준으로 안내된다.
- 계층별 최대 거주기간이 다르다.
- 소득·자산 기준과 무주택 기준을 함께 확인한다.

## 공급대상과 최대 거주기간

| 계층 | 대표 최대 거주기간 |
| --- | --- |
| 대학생 | 10년 |
| 청년 | 10년 |
| 산업단지근로자 | 10년 |
| 신혼부부 | 10년, 자녀가 있는 경우 14년 |
| 한부모가족 | 10년, 자녀가 있는 경우 14년 |
| 일자리연계형 지원주택 입주자 | 10년, 자녀가 있는 경우 14년 |
| 고령자 | 20년 |
| 주거급여수급자 | 20년 |

상담 주의:

- “행복주택은 무조건 10년”이라고 말하지 않는다.
- 계층과 자녀 여부에 따라 최대 거주기간이 달라진다.
- 실제 거주 가능 기간은 공고와 재계약 요건을 기준으로 확인한다.

## 계층별 기본 자격

| 계층 | 대표 자격 |
| --- | --- |
| 대학생 | 대학 재학 중 또는 입·복학 예정인 혼인 중이 아닌 무주택자 |
| 취업준비생 | 대학, 고등학교 또는 동등 학력 인정 교육기관을 졸업·중퇴한 지 2년 이내인 혼인 중이 아닌 무주택자 |
| 청년 | 만 19세 이상 만 39세 이하 혼인 중이 아닌 무주택자 |
| 사회초년생 | 소득 있는 업무 종사기간이 총 5년 이내인 혼인 중이 아닌 무주택자 |
| 신혼부부 | 혼인기간 7년 이내 또는 만 6세 이하 자녀를 둔 무주택세대구성원 |
| 예비신혼부부 | 혼인을 계획 중이며 입주 전까지 혼인사실을 증명할 수 있고, 혼인으로 구성될 세대원 모두 무주택자인 경우 |
| 한부모가족 | 만 6세 이하 자녀를 둔 무주택세대구성원 |
| 고령자 | 만 65세 이상 무주택세대구성원 |
| 주거급여수급자 | 무주택세대구성원인 주거급여 수급권자 또는 수급자 |

상담 주의:

- 청년과 대학생은 “무주택자” 기준으로 표현되는 경우가 있고, 신혼부부·고령자 등은 “무주택세대구성원” 기준으로 보는 경우가 많다.
- 예비신혼부부는 입주 전 혼인사실 증명이 핵심이다.
- 사회초년생은 종사기간, 퇴직 후 기간, 구직급여 수급자격, 예술인 여부 등 세부 기준을 확인한다.

## 소득 기준

대표 기준은 전년도 도시근로자 가구원수별 월평균소득의 100% 이하이다.

가구원 수 보정:

- 1인 가구: 120% 이하 기준 적용
- 2인 가구: 110% 이하 기준 적용
- 맞벌이 신혼부부: 120% 이하 기준 적용 가능

계층별 소득 산정:

- 대학생: 본인과 부모 소득을 함께 본다.
- 세대원인 청년: 본인 소득을 본다.
- 청년·사회초년생: 세대 구성 방식에 따라 본인 또는 해당 세대 기준을 확인한다.
- 신혼부부·한부모가족·고령자: 해당 세대 월평균소득 합계를 본다.

2025년도 도시근로자 월평균소득 기준 주요 금액은 다음과 같다.

| 가구원 수 | 100% | 110% | 120% |
| --- | ---: | ---: | ---: |
| 1인 | 3,813,363원 이하 | 4,194,699원 이하 | 4,576,036원 이하 |
| 2인 | 5,866,270원 이하 | 6,452,897원 이하 | 7,039,524원 이하 |
| 3인 | 8,168,429원 이하 | 8,985,272원 이하 | 9,802,115원 이하 |
| 4인 | 8,802,202원 이하 | 9,682,422원 이하 | 10,562,642원 이하 |
| 5인 | 9,326,985원 이하 | 10,259,684원 이하 | 11,192,382원 이하 |
| 6인 | 9,906,263원 이하 | 10,896,889원 이하 | 11,887,516원 이하 |

상담 주의:

- 가구원 수, 부양가족 수, 미성년 자녀 수에는 태아가 포함될 수 있다.
- 대학생·청년·신혼부부는 소득을 보는 범위가 다르므로 계층 확인이 먼저다.
- 소득 기준 금액은 공고 기준연도에 따라 달라질 수 있다.

## 자산 기준

2026년 기준 대표 자산 기준:

| 계층 | 총자산 기준 |
| --- | ---: |
| 대학생 | 10,800만원 이하 |
| 청년 | 25,100만원 이하 |
| 고령자 | 34,500만원 이하 |
| 산업단지근로자 | 34,500만원 이하 |
| 신혼부부·한부모가족 | 34,500만원 이하 |

자동차 기준:

- 자동차 가액 4,542만원 이하
- 대학생은 자동차가액 산출대상 자동차를 소유하지 않아야 한다.

상담 주의:

- 행복주택은 계층별 자산 기준이 다르므로 “행복주택 자산기준은 하나”라고 말하지 않는다.
- 대학생은 자동차 보유 제한이 더 엄격하게 적용될 수 있다.
- 자동차는 비영업용 승용자동차를 기준으로 하며, 장애인 사용 자동차와 국가유공자 보철용 차량 등은 제외될 수 있다.

## 입주 신청 절차

LH 청약플러스 행복주택 안내 흐름은 다음 순서다.

1. 신청: 현장 또는 인터넷 접수
2. 서류제출대상자 발표
3. 서류제출대상자 서류 접수
4. 소득·자산 조사: 사회보장정보시스템 활용
5. 소득·자산 소명 요청
6. 소명 접수 및 심사
7. 당첨자 발표

상담 적용:

- 신청만으로 당첨이 확정되는 것이 아니며 서류와 소득·자산 조사가 이어진다.
- 소명 요청을 받은 경우 기한 내 객관적 증빙자료 제출이 중요하다.
- 현장 접수 또는 인터넷 접수 여부는 공고별로 다를 수 있다.

## 임대조건

행복주택 임대보증금과 임대료는 각각 표준임대보증금과 표준임대료 이하에서 결정된다.

공급대상 계수는 다음과 같이 안내된다.

| 공급대상 | 공급대상 계수 |
| --- | ---: |
| 신혼부부, 한부모가족, 장기근속자 | 0.80 |
| 산업단지근로자 | 0.80 |
| 고령자 | 0.76 |
| 소득이 있는 청년 | 0.72 |
| 대학생, 소득이 없는 청년 | 0.68 |
| 주거급여수급자 | 0.60 |

임대보증금과 임대료는 상호전환이 가능할 수 있다. LH 청약플러스 안내상 전환단위는 100만원 단위이고, 전환이율은 현 기준 임대료에서 보증금으로 전환할 때 연 6.0%, 보증금에서 임대료로 전환할 때 연 3.5%로 안내된다. 실제 적용은 공고와 사업시행자 기준을 확인한다.

## 청주나 상담 응답 예시

### “행복주택이 뭐야?”

행복주택은 대학생, 청년, 신혼부부, 한부모가족 등 생애 초기 계층과 고령자·주거급여수급자의 주거비 부담을 낮추기 위해 공급하는 공공임대주택입니다. 대중교통이 편리하거나 직장·학교와 가까운 곳에 공급되는 경우가 많고, 계층별로 소득·자산 기준과 최대 거주기간이 다릅니다.

### “청년이면 행복주택 가능해?”

청년 행복주택은 보통 만 19세 이상 만 39세 이하 혼인 중이 아닌 무주택자를 대상으로 봅니다. 다만 소득, 총자산, 자동차 기준과 공고별 지역·순위 조건을 함께 확인해야 합니다. 최종 판단은 모집공고문 원문 기준으로 확인해야 합니다.

### “행복주택은 몇 년 살 수 있어?”

대학생·청년·산업단지근로자는 대표적으로 최대 10년, 신혼부부·한부모가족은 10년 또는 자녀가 있는 경우 14년, 고령자와 주거급여수급자는 20년까지 안내됩니다. 실제 거주기간은 계층, 자녀 여부, 재계약 조건과 공고 기준을 확인해야 합니다.

## Agent 적용 규칙

- Policy Agent: 행복주택의 정의, 공급대상, 국민임대와의 차이, 최대 거주기간을 설명한다.
- Eligibility Agent: 나이, 혼인 여부, 무주택 여부, 계층, 소득 산정 범위, 자산 기준을 먼저 확인한다.
- Recommendation Agent: 청년·대학생·신혼부부 등 사용자의 계층과 행복주택 공고의 공급대상을 비교한다.
- Verification Agent: 소득·자산 금액, 자동차 기준, 최대 거주기간은 공고 기준연도와 최신 LH 안내를 재확인한다.
