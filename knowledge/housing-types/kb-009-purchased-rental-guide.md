# KB-009 매입임대 가이드

- KB ID: `KB-009`
- 주제: 매입임대 상담 기준
- 기준일: 2026-07-03
- 사용 범위: Policy Agent, Eligibility Agent, Recommendation Agent, Verification Agent
- 우선순위: 모집공고 원문 > LH 청약플러스 최신 안내 > 이 문서

## 한 줄 정의

매입임대는 LH 등이 기존 주택을 매입한 뒤 저소득층, 청년, 신혼부부, 다자녀, 고령자 등에게 시세보다 저렴하게 다시 임대하는 주택이다.

청주나 상담에서는 “공공기관이 집을 사서 대상자에게 싸게 다시 빌려주는 방식”으로 설명한다.

## 주요 세부 유형

| 유형 | 핵심 설명 |
| --- | --- |
| 일반 매입임대 | 도심 내 저소득계층이 현 생활권에서 거주하도록 LH가 매입한 주택을 시세 30% 수준으로 임대 |
| 청년 매입임대 | 청년, 대학생, 취업준비생 대상. 시세 40~50% 수준 |
| 신혼·신생아 매입임대Ⅰ | 신생아가구, 신혼부부 등 대상. 시세 30~40% 수준 |
| 신혼·신생아 매입임대Ⅱ | 전세형 성격. 시세 70~80% 수준 |
| 다자녀 매입임대 | 두 명 이상의 직계비속을 양육하는 다자녀 가구 대상. 시세 30~40% 수준 |
| 고령자 매입임대 | 저소득 고령자층 대상. 시세 40% 수준 |
| 든든전세주택 | 넓고 쾌적한 신축주택을 매입해 중산층 가구에게 시세 90% 이하 수준으로 임대 |
| 기숙사형 매입임대 | 대학생 등에게 시세 40% 수준으로 임대 |
| 집주인 임대 | 민간 소유 주택을 LH에 위탁임대해 청년·고령자·신혼부부 등에게 저렴하게 공급 |

## 상담 주의

- 매입임대는 하나의 자격기준으로 판단할 수 없고, 세부 유형별 대상과 소득·자산 기준이 다르다.
- 청년형, 신혼·신생아형, 다자녀형, 고령자형을 먼저 구분해야 한다.
- 실제 주택 위치와 면적, 임대조건은 매입 주택별로 차이가 크다.

## 청주나 상담 응답 예시

### “매입임대는 뭐야?”

매입임대는 LH 같은 공공기관이 기존 주택을 매입한 뒤 청년, 신혼부부, 저소득층, 고령자 등에게 시세보다 저렴하게 다시 임대하는 방식입니다. 청년 매입임대, 신혼·신생아 매입임대, 다자녀 매입임대처럼 세부 유형이 나뉘므로 먼저 어떤 대상 유형인지 확인해야 합니다.

## Agent 적용 규칙

- Policy Agent: 매입임대의 구조와 세부 유형을 설명한다.
- Eligibility Agent: 대상 유형, 나이, 혼인·자녀, 소득·자산, 무주택 여부를 확인한다.
- Recommendation Agent: 사용자의 생애계층에 맞는 매입임대 세부 유형을 우선 매칭한다.
- Verification Agent: 세부 유형별 임대료 수준과 신청자격은 공고 원문을 확인한다.
